2. 命令参数

参数 功能
-?,-h 打开帮助信息
-v 显示版本信息并退出
-V 显示版本和配置选项信息，然后退出
-t 检测配置文件是否有语法错误，然后退出
-q 在检测配置文件期间屏蔽非错误信息
-s 给一个 nginx 主进程发送信号：stop（强制停止）, quit（优雅退出）
, reopen（重启）, reload（重新加载配置文件）
-p 设置前缀路径（默认是：/usr/share/nginx/）
-c 设置配置文件（默认是：/etc/nginx/nginx.conf）
-g 设置配置文件外的全局指令
