1. 偏函数

所谓偏函数，就是固定一个函数的一个或者多个参数，返回一个新的函数，这个函数用于接受剩余的参数。

2. bind 方法

由于 bind 函数会返回一个未执行的新函数，且新函数与被调函数具有相同的函数体，所以我们也可以使用 bind 方法来实现偏函数

2. 函数柯里化

柯里化是另一个有趣的处理函数技术。

柯里化（Currying）：转换一个调用函数f(a,b,c)为f(a)(b)(c)方式调用。
